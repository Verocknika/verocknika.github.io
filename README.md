# 🦊 Personal Portfolio Project – Veronika

> **Note:** This project is a fusion of a school assignment and a personal digital portfolio, bridging the gap between academic requirements and professional branding.

---

## 💡 Concept & Philosophy

This website is more than just a digital CV. It is built on the belief that **nature and outdoor challenges** shape a person's character. By showcasing mountain expeditions and personal hobbies, the project highlights essential "soft skills" transferred into the professional world:

* **Discipline & Focus**
* **Analytical Thinking & Logic**
* **Adaptability in changing environments**
* **Responsibility & Precision**

---

## 🛠️ Tech Stack

The project is built as a lightweight, high-performance static site:

* **HTML5:** Semantic structure for optimal SEO and accessibility.
* **CSS3:** Custom styling featuring:
* **Flexbox & CSS Grid:** For a fully responsive layout.
* **Interactive Hover Effects:** Revealing insights behind the imagery.
* **Modern UI:** Clean typography and a nature-inspired color palette.


* **JavaScript:** Minimalist scripts for mobile navigation (Hamburger Menu).
* **Formspree:** Integration for a functional, secure contact form.

---

## 📁 Website Architecture

The site is organized into five logical sections:

| Section | File | Description |
| --- | --- | --- |
| **Home** | `index.html` | Introduction and call to action. |
| **About Me** | `omne.html` | An interactive grid linking life lessons to photography. |
| **Experience** | `skusenosti.html` | A professional journey (currently in progress). |
| **Skills** | `znalosti.html` | Technical expertise: Python, SQL, Linux, and Blender. |
| **Contact** | `kontakt.html` | Direct communication channel with file upload support. |

---

## 🚀 Key Features

* 📱 **Fully Responsive:** Seamless experience across mobile, tablet, and desktop.
* 📸 **Visual Storytelling:** A unique approach to presenting professional attributes through hobby-based metaphors.
* 🧭 **Intuitive Navigation:** Sticky header for easy access to all sections.
