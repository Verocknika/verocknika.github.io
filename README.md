Personal Portfolio Project – Veronika
This project is a static website that serves as both a school assignment and a personal digital portfolio. The goal was to combine technical requirements with an authentic presentation of my values, professional skills, and personal interests.

💡 Concept
The website goes beyond a traditional CV. It is built on the philosophy that mountains and outdoor experiences shape character, translating vital "soft skills"—such as discipline, analytical thinking, and adaptability—directly into the professional world.

🛠️ Technologies Used
HTML5: Semantic structure for all pages.

CSS3: Custom styling including:

Flexbox & Grid: Used for layout and responsiveness.

Interactive Hover Effects: Feature used in the "About Me" section to reveal content.

Modern UI: Focus on readability and a clean aesthetic.

JavaScript: Simple scripts to handle the mobile navigation toggle.

Formspree: Integration for a functional contact form with file upload capability.

📁 Website Structure
The project consists of five main sections:

Home (index.html): A brief introduction and a call to action.

About Me (omne.html): An interactive grid where each photo represents a life lesson or trait, such as perseverance, responsibility, or aesthetics.

Experience (skusenosti.html): A timeline of my professional journey.

Skills (znalosti.html): A breakdown of technical expertise in IT (Python, SQL, Linux), data analysis, and 3D graphics (Blender).

Contact (kontakt.html): A professional form for direct communication.

🚀 Key Features
Responsive Design: Optimized for both mobile devices and desktops.

Visual Storytelling: Using photography to bridge the gap between hobbies and professional attributes.

Clean Navigation: A sticky navbar for seamless browsing across the site.

Note: This project is under continuous development, with certain sections like "Experience" currently being finalized.
